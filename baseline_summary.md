# Baseline Summary

This file is included as a plain-language baseline summary, but the public retrieval-strategy evaluator does not score it directly.

The demo framework contains:

- durable state memories
- correction memories
- uncertainty memories
- failure-recovery memories
- authority-policy memories
- access-policy memories

Known limitations:

- internal diagnostics are not proof
- private-origin claims require verification
- summaries are not final source of truth when live files disagree
- external rewrites should not replace authorial voice
- access policy can answer, provide context, warn, verify first, block, or archive

Use the structured memory files for the actual deterministic retrieval evaluation.
